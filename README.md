# 有道云图书 · 编辑器（Vue3）

一个参考有道云笔记交互的图书管理与写作工作台，基于 **Vue 3 + Pinia + Vite** 构建，内置 Markdown 编辑与实时预览、灵感收藏、标签筛选等功能，适合用来演示知识型图书的编辑流程。

## 主要特性
- 📚 **多维图书库视图**：按收藏、草稿、归档等视角快速切换，并支持关键字、状态、标签多条件过滤。
- 🧠 **Pinia 状态管理 + localStorage 持久化**：离线也可继续写作，自动保存最近一次打开的图书。
- ✍️ **富文本写作体验**：Markdown 编辑区域与预览区域并排展示，实时同步，适配有道云的排版风格。
- 🧾 **元数据面板**：即时维护作者、分类、状态、标签与摘要，方便后续检索和成稿管理。
- ⚡ **便捷操作**：收藏、复制、删除、一键新建等动作都在列表卡片即可触达。

## 本地运行
```bash
npm install
npm run dev
```

默认启动在 `http://localhost:5173`，可通过环境变量自定义端口。

## 构建生产版本
```bash
npm run build
npm run preview # 可选：本地预览打包产物
```

## 目录结构
```
├─ src
│  ├─ components    # 侧边栏、书籍列表、编辑器等 UI 组件
│  ├─ data          # 初始示例图书数据
│  ├─ stores        # Pinia 图书仓库，封装增删改查与过滤逻辑
│  ├─ utils         # 时间格式化等工具函数
│  ├─ App.vue       # 布局与页面骨架
│  └─ main.js       # 入口文件
└─ vite.config.js   # Vite 配置
```
